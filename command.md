2. 필요 내용 아래 인터페이스로 로그를 남길거임.
   1. Info
   2. Error
   3. Warning
4. 근데 해당 로직은 싱글턴 및 스레드로 돌면서 작업진행하는 상황
5. 함수 호출시엔 Queue에 적재 스레드에서는 Queue에서 하나씩 빼서 파일로 남기고 콘솔에 찍도록.
6. 추가로 로그를 남길때 타임스탬프도 같이 남기도록. 
   1. YYYY-MM-DD HH:MM:SS|로그 형식
   2. 각각의 인터페이스에 대해서 파일을 생성함.
   3. 날짜가 다음날로 넘어가면 새로운 파일을 생성해서 로그를 남기도록.
   4. 해당 string을 다룰때 최대한 성능저하가 없도록 다루도록.
7. 큐가 비면 스레드가 대기하도록.
